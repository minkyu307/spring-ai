# Overview

This project is built using Spring Boot 3.5.9, Java 21, Spring Google Gen AI 1.1.2, Lombok, Spring Data JPA, and Thymeleaf.

For official documentation on Spring AI, refer to URLs starting with “https://docs.spring.io/spring-ai/reference/”.
